# Auto Trip Guide

## Web App
The application will be used to add sites to the system by authorized persons.

The application includes:
* **Search:** Search for pois by categories.
* **Add:** Add new poi.
* **Edit:** edit exist poi.
* **Delete:** Delete exist poi.
* **Approve:** Approve exist poi.
